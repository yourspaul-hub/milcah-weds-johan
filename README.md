# Milcah & Johan — Wedding Invitation

A single-page wedding invitation site built with Next.js 15, TypeScript, Tailwind CSS and Framer Motion.

## Features
- Animated envelope open/tap-to-continue intro
- Watercolor illustration background (where the couple first met)
- Auto-playing background music with a floating player control
- Live countdown to the wedding day
- Order of ceremony timeline
- Google Maps "Get Directions" links
- Sage Green / Ivory / Dusty Rose / Champagne / Antique Gold / Soft Taupe color palette

## Getting started
```bash
npm install
npm run dev
```

Edit `lib/wedding-config.ts` to change names, dates, venue, and messages.
